## ✨ 线程执行顺序控制的业务场景


### 场景描述

- A： 用户开通会员可以使用券抵扣部分资金
- B：通知活动系统扣减券冻结
- C：通知额度系统进行额度冻结
- D：对用户进行扣款处理
- E：开通会员成功

假设这些步骤都是同步成功的。

现在的条件是 B、C、D 这3个流程没有关联性，可以同时处理；但是步骤E执行时要求收到B、C、D的处理是顺序发生的。

可以使用 Thread.join方法控制顺序执行的方式。
